# Stopwatch Application

A professional stopwatch application built with React, TypeScript, and shadcn-ui. Features precision timing with lap tracking, fastest/slowest lap detection, and a modern, responsive interface.

## Getting Started

This application can be edited using your preferred IDE.

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh

git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd <YOUR_PROJECT_NAME>

# Step 3: Install the necessary dependencies.
npm i

# Step 4: Start the development server with auto-reloading and an instant preview.
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Features

- **Precision Timing**: Uses requestAnimationFrame for accurate time tracking
- **Lap Recording**: Record multiple laps with automatic numbering
- **Performance Analysis**: Automatically identifies fastest and slowest laps
- **Modern UI**: Built with shadcn-ui components and Tailwind CSS
- **Responsive Design**: Works seamlessly on desktop and mobile devices
